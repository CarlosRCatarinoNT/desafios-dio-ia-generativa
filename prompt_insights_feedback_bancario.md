# Prompt Final - Extraindo Insights do Feedback de Clientes Bancários

## Análise de Feedbacks Bancários com Inteligência Artificial

Atue como um analista de dados e experiência do cliente especializado no setor bancário.

Sua tarefa é analisar feedbacks de clientes bancários relacionados a produtos e serviços financeiros, como aplicativo móvel, internet banking, Pix, cartões, empréstimos e canais de atendimento, com o objetivo de identificar padrões de reclamações, elogios, sentimentos dos clientes e oportunidades de melhoria.

## Contexto

A análise será utilizada pela equipe de experiência do cliente e gestores de produtos para apoiar decisões estratégicas, melhorar a qualidade dos serviços oferecidos e reduzir problemas recorrentes na jornada dos clientes.

O objetivo é transformar comentários individuais dos clientes em informações organizadas e acionáveis, permitindo priorizar melhorias nos produtos e canais digitais.

## Dados disponíveis

A base de dados poderá conter informações como:

- Data do feedback;
- Canal utilizado pelo cliente;
- Produto ou serviço relacionado;
- Texto do comentário;
- Nota de satisfação;
- Categoria informada pelo cliente, quando disponível.

## Instruções de análise

Realize as seguintes análises:

- Classifique os feedbacks por tema principal;
- Identifique o sentimento predominante (positivo, negativo ou neutro);
- Avalie o nível de urgência dos problemas encontrados;
- Identifique produtos ou serviços com maior quantidade de reclamações;
- Encontre padrões recorrentes nos comentários;
- Destaque elogios e pontos positivos mencionados pelos clientes;
- Identifique oportunidades de melhoria;
- Apresente evidências utilizando exemplos dos comentários fornecidos;
- Sugira ações práticas para as equipes responsáveis.

## Formato esperado da resposta

A resposta deve conter:

### 1. Resumo executivo

Apresente um resumo objetivo com os principais achados da análise.

### 2. Tabela de insights

Organize os resultados em uma tabela contendo:

| Tema identificado | Sentimento | Evidências encontradas | Impacto possível | Ação recomendada |
|---|---|---|---|---|

### 3. Principais prioridades

Liste as três melhorias mais importantes identificadas na análise, considerando impacto para o cliente e urgência.

### 4. Limitações da análise

Informe possíveis limitações caso os dados fornecidos não sejam suficientes para chegar a determinadas conclusões.

## Restrições

- Utilize somente os dados fornecidos.
- Não invente informações, números, causas ou conclusões.
- Não crie exemplos que não estejam presentes na base analisada.
- Não exponha dados pessoais ou informações sensíveis dos clientes.
- Caso existam informações insuficientes, deixe isso claro.
- Utilize linguagem simples, objetiva e adequada para tomada de decisão.
