# 🚀 Desafios DIO - Inteligência Artificial Generativa

Repositório criado para armazenar desafios práticos realizados durante minha jornada de aprendizado em Inteligência Artificial Generativa pela DIO.

## 📚 Objetivos

Este repositório tem como objetivo documentar exercícios relacionados a:

- Engenharia de Prompts;
- Uso responsável de Inteligência Artificial;
- Criação de instruções eficientes para modelos de IA;
- Aplicação prática de IA na análise de informações.

## 📂 Desafios

### 🎯 Extraindo Insights do Feedback de Clientes Bancários

Desafio focado na criação de um prompt estruturado para orientar uma IA na análise de feedbacks de clientes bancários.

O objetivo foi definir:

- Contexto da análise;
- Critérios de classificação;
- Restrições para evitar respostas incorretas;
- Formato esperado dos insights gerados.
